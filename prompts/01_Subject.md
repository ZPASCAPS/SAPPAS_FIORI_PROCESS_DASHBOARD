# 01. Project Topic Prompt

이 프로젝트는 SAP 사용자를 위한 통합 업무 프로세스 대시보드이다.

SAP 업무 프로세스의 흐름을 시각화하는 실무형 Fiori 대시보드를 만든다.

## 프로젝트 핵심 주제

SAP Fiori 기반 통합 프로세스 대시보드를 구현한다.

메인 대시보드는 다음 구조를 가진다.

1. 왼쪽 영역
   - 세부 기능을 선택할 수 있는 모듈별 탭 또는 사이드 메뉴
   - SD, MM, PP, Inventory, Procurement, Decision 등의 기능을 선택할 수 있게 구성

2. 중앙 영역
   - SAP 업무 프로세스 흐름을 순서도로 표시 (메인 대시보드)
   - 데이터 시각화 및 중요 기능 (왼쪽 탭 - 기능 대시보드)

3. 오른쪽 영역
   - 사용자가 질문할 수 있는 챗봇 또는 AI Assistant 패널 배치 (메인 대시보드)

## 핵심 업무 흐름


## 개발 방향

- SAP Fiori / SAPUI5 스타일을 유지한다.
- 사용자가 업무 흐름을 직관적으로 이해할 수 있게 만든다.
- 과도한 장식보다 실무형 화면 구성을 우선한다.
- 추후 SAP OData 연동이 가능하도록 구조를 설계한다.
- 처음에는 JSONModel 더미데이터로 구현하고, 이후 실제 SAP OData 데이터로 교체할 수 있게 만든다.
